Projeto Cosso Library

Esse projeto tem como foco imitar um Sistema Controle de Livros em Livraria virtual Cosso Library, onde mostra todos os livros e categorias que existem nela.

O projeto foi desenvolvido por Vitor Cosso durante os meses de Dezembro de 2023 e Janeiro de 2024.

Tecnologias e Práticas

As tecnologias utilizadas foram Java Spring Boot, Angular 17, H2 Database (Banco de dados em Memoria que utiliza a mesma sintaxe do MySQL), MySQL em testes, Git e Github, Visual Code Studio, Clean Code e Arquitetura Limpa.

Foco

Esse é um projeto pessoal com foco em aprendizado e dominío das linguaguens, tecnologias e praticas utilizadas durante o desenvolvimento.
